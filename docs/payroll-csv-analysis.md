## 実装時の注意点

### 1. 同名列がある
給与CSVには `保育海外研修` が 2 回出てくる。
- 前半は支給項目
- 後半は控除項目

CSV取込時はヘッダー名だけで map を作ると上書きされるので、列インデックスも保持すること。

### 2. 全列が明細表示に必要とは限らない
明細PDFでは次の3ブロックで組むのが自然。
- 支給項目一覧
- 控除項目一覧
- 集計欄

`0` の項目は非表示にして見やすくしてよい。

### 3. 金額計算はCSV値を正とする
給与計算ロジックをサーバー側で再計算するより、CSVの金額をそのまま明細表示に使う方が安全。

### 4. 追加で必要な管理情報
CSV自体には次の運用情報が入っていない可能性があるため、アップロード時に別入力が必要。
- 対象年月
- 明細種別: `PAYROLL` または `BONUS`
- 公開日時

- タイトル: `給与明細`
- 対象者: `社員番号`, `姓`, `名`
- 支給欄: 支給項目を上から表示
- 控除欄: 控除項目を上から表示

  - `振込支給`
