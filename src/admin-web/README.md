# 管理Web試作

実装先:

- 静的試作: `src/admin-web/prototype`
- Next.js 本実装骨組み: `src/admin-web/next-app`

## 内容

- ダッシュボード
- 職員一覧
- カード管理
- 打刻履歴
- 休暇申請承認
- 給与明細アップロード
- 監査ログ一覧

## 特徴

- 依存ライブラリなしで開ける静的プロトタイプです
- バックエンドのモックAPIへ `fetch` で接続します
- 後から Next.js 実装へ移し替える際の画面たたき台として使えます
- `next-app` 側には App Router ベースの本実装骨組みを追加済みです

## 使い方

ブラウザで `index.html` を開き、必要なら画面右上で API Base URL を変更してください。
